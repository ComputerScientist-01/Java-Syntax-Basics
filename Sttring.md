# String
Creation:
```java
String gopha = new String(“ok”);
```

String literal:
```java
String gopha = “ok”;
```

Size:
```java
gopha.length();
```

Accessing:
```java
char[] chArr = gopha.toCharArray();
for (char c : chArr) 
    { System.out.print(c); }

for (int i = 0; i < gopha.length(); i++) { 
    System.out.print(gopha.charAt(i)); }
```
